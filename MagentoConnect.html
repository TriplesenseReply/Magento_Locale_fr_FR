Französisches Sprachpaket für Magento Commerce

(Magento Community Modules - French (France) Language Pack)

(Paquete de idioma francés para Magento Commerce)

Wichtiger Hinweis zur Magento Version 1.9

Magento Inc. hat mit der 1.9 ein eigenes, teilautomatisch übersetztes, Sprachpaket vorinstalliert. Wer das LocalePack_fr_FR benutzen möchte, muss vorher den Ordner app/locale/fr_FR löschen, damit dieses Sprachpaket installiert werden kann.

Nota importante acerca Magento Versión 1.9

Magento Inc. tiene con la versión 1.9 su propio, semi-automáticamente traducido, paquete de idiomas como preinstalado. El que quiere usar el LocalePack_fr_FR, primero debe eliminar la carpeta app/locale/fr_FR para que el paquete de idioma se puede instalar.
